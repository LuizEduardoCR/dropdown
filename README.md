# Cinemateca

Bem-vindo à Cinemateca! 

## Como Executar o Projeto

1. Clone o repositório:
    ```bash
    git clone https://github.com/mariaeduardanichelle/cinemateca.git
    ```
2. Navegue até o diretório do projeto:
    ```bash
    cd cinemateca
    ```
3. Instale as dependências:
    ```bash
    npm install
    ```
4. Inicie o servidor:
    ```bash
    npm run dev
    ```
